## 📖 Sobre o Projeto

Este é um projeto pessoal, nascido de uma ideia original para solucionar um problema clássico de todo colecionador: **a organização das figurinhas do álbum da Copa do Mundo**. 

A proposta desta aplicação é oferecer uma interface simples e intuitiva onde o usuário possa controlar seu acervo de figurinhas, facilitando não apenas a visualização do que falta para completar o álbum, mas também o gerenciamento de figurinhas repetidas para auxiliar nas trocas com os amigos.

## ✨ Funcionalidades

- **Controle de Acervo:** Marque facilmente quais figurinhas você já colou no álbum (possuídas) e quais ainda faltam.
- **Gestão de Repetidas:** Uma área dedicada para marcar as figurinhas duplicadas (repetidas), tornando a vida do colecionador muito mais fácil na hora de organizar trocas.
- **Navegação por Seleções:** Interface organizada por grupos e seleções, permitindo encontrar a figurinha desejada rapidamente.
- **Interface Responsiva:** (Opcional: Mantenha se o seu projeto funcionar bem no celular) Design moderno adaptado para funcionar bem tanto no computador quanto no celular.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais (Vanilla):
- **HTML5** - Estruturação do aplicativo.
- **CSS3** - Estilização, layout e animações (com fontes personalizadas da família *Barlow*).
- **JavaScript** - Lógica de manipulação do DOM e controle dos estados das figurinhas (`missing`, `owned`, `repeated`).
